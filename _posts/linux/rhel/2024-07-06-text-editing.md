## 1. Command List

* `vi spiderfile` : Opens file with vi editor.
* `i` : Gets to insert mode starts writing.
* `o` : Gets to insert mode starts writing with new line up.
* `a` : Gets to insert mode starts writing with new line bottom.
* `esc` : Gets out from insert mode.
* `:wq!` : Closes vi editor with write quit force.

-----------------------------------------------------

* `grep spiders spiderfile` : Returns single line with filtering.
* `:/spiders` : Searches spiders in command mode of vi.
