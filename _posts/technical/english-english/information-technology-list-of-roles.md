## Software Developer

First of all, he/she is expected to have command of the programming language.
Then, he/she is expected to be knowledgeable about the frameworks of this language.
Able to develop independently of Fullstack and software language.

For example for Java language Hibernate, Spring and Spring Boot frameworks.
SQL (MySQL, PostgreSQL, Oracle, etc.) and NoSQL (Cassandra, Clickhouse, etc.) database technologies
Proficient in synchronous system architectures
OOP concepts
web services such as SOAP, XML, RPC, REST
version control tools (SVN, CVS, GIT etc.)
Familiarity with Linux
Redis, Elasticsearch, RabbitMQ, Kafka

* Analyzing, developing and redesigning existing applications or new applications when necessary.
* Implementing software testing processes to ensure software quality.
* To develop high-quality and scalable software solutions by using programming languages ​​and best practice standards, focusing on performance, security and usability.
* Using tools and methodologies to automate and improve continuous integration and continuous delivery processes.
* Participating in code reviews, having the written codes reviewed and following the processes.
* Documenting code and systems in an understandable way, creating documentation to facilitate the work of other developers and stakeholders.
* Taking part in different project management methodologies such as Agile and Waterfall, communicating with testing, development and analysis teams within the organization or with the customer and taking an active role in the progress of the work.
* To discover, learn and apply different technologies and software languages ​​specific to needs.
