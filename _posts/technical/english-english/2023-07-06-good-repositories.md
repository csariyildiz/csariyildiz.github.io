List of good repositories on Github to explore consists of various repositories.

* [PenEditor](https://github.com/jojowwbb/PenEditor). Code editor written in React. Uses bulma.css and [codemirror](https://github.com/codemirror/dev/)
* [Flask For Startups](https://github.com/nuvic/flask_for_startups). This flask boilerplate.
* [OpenAI Chat Window](https://github.com/ahutanu/openai-chat-window/). A Sample OpenAI Project. A playground chat which you can use to play with OpenAI's Text Completion Models
* [Rustdesk](https://github.com/rustdesk/rustdesk). Remote desktop software, written in Rust.
* [Twitters Bootstrap](https://github.com/twbs/bootstrap).
* [Jquery](https://github.com/jquery/jquery). Public repository of jQuery JavaScript library.
* [Datatables](https://github.com/DataTables/DataTables). The table enhancing plug-in for jQuery.
* [Sample Curriculum For cyber security](https://github.com/DerekBabb/CyberSecurity). A curriculum designed for a computer science course focused on cyber security. 
* [Fluent Reader](https://github.com/yang991178/fluent-reader)
