Latex is definitely a nice, professional way to write. It is widely used in academia. But sometimes it can be difficult for beginners to use.
So I want to focus on latex and its usage.

I tried MikTex with TextView but not find it very user friendly. 
Problems arose when the foundation was a little worn. One reason for this was probably because I was using the Windows operating system.

At this point, I quite liked overleaf, which is a cloud solution. overleaf is very successful and seems to have closed a major gap although I have found plenty of quirks in its documentation at times and raises concerns about the security of the scripts.
We can use Overleaf for free. I used this system in the rest of the document.
