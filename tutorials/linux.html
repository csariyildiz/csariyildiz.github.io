Linux System Administration Notes

Hi! These are Linux operating system notes are organized in commands notes and topics. Mainly on RHEL9 operating system but not limited by it.

  • Commands
  • Notes
  • Topics

Commands are written in a form of list of commands on that subject. Additional text notes are quick overview of conceptsand topics are generally for detailing some commands outputs or concepts.

Notes are created using RHCSA exam objectives along with course series on Udemy and with the help of chatbots. Care has been taken not to directly use materials with copyrights.

They are organized in commands, notes and topics for each section. This way I tried to keep them more practice oriented. Each section has its commands, notes and topic section. If a topic is not practice oriented (such as explaining linux kernel) than I share them in additional notes section. Commands are created for using with flashcards.

Tips

Whether you want to specialize in the subject or are preparing for the exam, the generally recommended tips are as follows.

  • Practice with virtual machines: Set up virtual machines like VirtualBox to experiment with network configurations in a safe environment.
  • Focus on memorization: Make flashcards for key commands, options, and configuration file parameters.
  • Focus on understanding: Aim to understand the functionality of each command rather than just memorizing syntax. This will help you apply your knowledge in different scenarios.
  • Creating checklists: Make checklists using any educational material and evaluate covered tasks with requirements.
  • Simulate exam conditions: Practice working on network configuration tasks under timed conditions to get used to exam pressure.
  • Review past exam papers (if available): Familiarize yourself with the types of questions asked in previous exams to tailor your studying accordingly.
