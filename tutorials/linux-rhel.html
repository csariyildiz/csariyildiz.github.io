Hi! These are Linux operating system notes are organized in commands notes and topics. Mainly on RHEL9 operating system but not limited by it.

Commands are written in a form of list of commands on that subject. Walktrough and notes for each section added after the listing of the commands.

Notes are created using RHCSA exam objectives along with course series on Udemy and with the help of chatbots. Official RHEL 9 documentation could be regarded as main resource when it comes to RHEL. Care has been taken not to directly use materials with copyrights. Documents widened by explaining details borrowing the existing concept from resources.

They are organized in commands, walktrough and details for each section. This way I tried to keep them from more practice oriented to detailed. It maybe beneficial to mostly memorizing the command part and go over walkthrough.

Tips

Whether you want to specialize in the subject or preparing for the exam, the generally recommended tips are as follows:

  • Practicing with virtual machines: Set up virtual machines like VirtualBox to experiment with network configurations in a safe environment.
  • Focusing on memorization: Make flashcards for key commands, options, and configuration file parameters.
  • Focusing on understanding: Aim to understand the functionality of each command rather than just memorizing syntax. This will help you apply your knowledge in different scenarios.
  • Creating checklists: Make checklists using any educational material and evaluate covered tasks with requirements.
  • Simulating exam conditions: Practice working on network configuration tasks under timed conditions to get used to exam pressure.
  • Reviewing past exam papers (if available): Familiarize yourself with the types of questions asked in previous exams to tailor your studying accordingly.
